Introduction

Bootstrap

Bootstrap is one of the easiest and most useful open source toolkits used to build websites reletively fast. It already contains almost every code and, we as developers just need to design our website by knowing the codes of Bootstrap. Also, it is possible to build a website without creating a CSS by just using HTML.

Bootstrap CSS

In bootstrap, CSS codes are already developed in the source. If you want to build a website by just using HTML, copy and paste the link from getbootstrap (under the CSS section) in the head section of your html and start building!

Responsive

Nowadays, responsiveness is key in website design. It is important to know that the best way to start building a website is from the mobile view. Bootstrap has built-in responsive codes in its source, copy and paste the link from getbootstrap (under the Responsive meta tag section) in the head section of your html and make your website responsive!

History

Twitter

Look at the first posts of how the bootstrap was created through twitter.

Read More

Bootstrap was originally released on August 19, 2011 and it was developed by Mark Otto and Jacob Thornton at twitter. This development is usually refered to a self-starting process and still today developers have been adding new features and updating new versions. According to Twitter developer Mark Otto:

"A super small group of developers and I got together to design and build a new internal tool and saw an opportunity to do something more. Through that process, we saw ourselves build something much more substantial than another internal tool. Months later, we ended up with an early version of Bootstrap as a way to document and share common design patterns and assets within the company." - wikipedia/Bootstrap
